## [1.0.0] - 2021.02.24
### Added
- Add changelog for this project.
- Add prop.go in prop directory and write some test.
- Add food global info notify.
- Add new file constants.go in configs directory to store some constants.



### Changed
- Modify some receiver's name from "this" to the first letter of its receiver(eg. gameRoom -> g).
- Change some file names and variable names to conform with [golang naming convention](https://golang.org/doc/effective_go). For example, change  GameRoomManager.go to game_room_manager.go.
- Update test_client.go in framework/test.
- Move serverAddr in main.go to configs/constants.go.
